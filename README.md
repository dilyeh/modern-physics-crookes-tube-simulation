# crookes tube simulation

## Dependencies
- Python 3.x
- pip3

## Installation

1. Clone git repo

```bash
git clone https://github.com/dilyeh/modern-physics-crookes-tube-simulation.git
```

2. Set up virtual environment and install dependencies

```bash
cd path/to/git/repo
python3 -m venv .venv
.venv/bin/activate
pip install -r requirements.txt
```

3. Run `thing.py`

```bash
python3 thing.py
```
